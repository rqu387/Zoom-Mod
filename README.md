# Zoom Mod for Minecraft 1.21.11 (Fabric)

## Installation

1. Make sure you have **Minecraft 1.21.11**, **Fabric Loader** and **Fabric API** installed.
2. Download the mod `Zoom 1.21.11fb.jar` file.
3. Place the `Zoom 1.21.11fb.jar` file into your `mods` folder (located in your Minecraft game directory). Create the `mods` folder if it doesn't exist.
4. Launch the game using the **Fabric** profile.
5. The mod is now active. Enjoy!

## How to Zoom

- Press and **hold** the Zoom key (default is usually `Z`).
- While holding the Zoom key, **scroll your mouse wheel** up/down to increase or decrease the zoom level.
- Release the key to return to normal view.

## Changing the Keybind (Optional)

To change the default Zoom key:

- Go to `Settings` → `Options` → `Controls` → `Key Binds`
- Find the **Miscellaneous** category
- Look for the Zoom mod entry (`rqu Zoom`)
- Click on it and press your desired key

**Tip:** You can also change the mouse scroll behavior in the mod's config file if available.

Good luck! :P
